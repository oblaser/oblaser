# Visual Studio Code User Settings

Path:
- `%APPDATA%\Code\User\settings.json`
- `~/.config/Code/User/settings.json`

```jsonc
{
    // default stuff...

    "window.restoreWindows": "none",
    "editor.rulers": [120],
    "[git-commit]": { "editor.rulers": [50] },
}
```
